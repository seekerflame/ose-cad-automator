### What is OSE CAD Automator?

OSE CAD Automator transforms FreeCAD CAD files into **platinum-quality build instructions** that anyone can follow to construct real-world projects.

## OSE:CAD/Quick_Start

### Prerequisites

- FreeCAD 0.21+ ([Download](https://www.freecad.org/downloads.php))
- Python 3.8+

### Step 1: Install

Run the one-click installer for your platform (see OSE:CAD/Automator).

### Step 2: Process Your First File

```bash
ose-cad process ~/Downloads/SH7-Floor-Module-5.fcstd
```

### Step 3: View Results

Open the generated `SH7-Floor-Module-5_Instructions.md` in any markdown viewer.

### Step 4: Build

Follow the step-by-step instructions to construct your project.

## OSE:CAD/Schema

### LOD 1000 Metadata Specification

The CAD Automator extracts metadata at "Level of Detail 1000" - enough information to understand assembly without full geometric data.

#### JSON Structure

```json
{
  "filename": "SH7 - Floor Module 5.fcstd",
  "parts": [
    {
      "name": "Box",
      "label": "2x8 Joist - rim 001",
      "type_id": "Part::Box",
      "properties": {
        "Label": "2x8 Joist - rim 001",
        "Placement": "Placement [Pos=(0,0,19.05), Yaw-Pitch-Roll=(0,0,0)]"
      }
    }
    // ... more parts
  ]
}
```

#### Supported Part Types

| Type | Description |
|------|-------------|
| `Part::Box` | Rectangular solid (lumber, panels) |
| `Part::Cylinder` | Cylindrical solid (pipes, holes) |
| `Part::Feature` | Compound/derived geometry |
| `App::DocumentObjectGroup` | Logical grouping |

## Contribution Workflow

### For Builders (Read-Only)

1. Visit any `OSE:CAD/*` page
2. Download the `.fcstd` source file
3. Process locally with `ose-cad`
4. Build your project!

### For Contributors (Edit Access)

1. **Fork** the CAD file from the protected page
2. **Edit** in FreeCAD
3. **Process** with `ose-cad` to verify
4. **Submit** via Talk page or mailing list
5. **Review** by maintainer
6. **Merge** to protected main page

### Protection Levels

| Page Type | Protection | Who Can Edit |
|-----------|------------|--------------|
| Core templates | Full | Maintainers only |
| Module pages | Semi | Registered users + review |
| Community | None | Anyone |
